## Overview

SEFARI's Operations module gives your team complete control over every delivery, service call, or order fulfillment — in real time.

## Dispatch

SEFARI's dispatch engine supports both manual and AI-automated assignment:

- **Manual dispatch** — Drag-and-drop orders onto drivers from the live map view
- **Auto-dispatch** — Rules-based assignment by zone, vehicle capacity, or driver availability
- **AI routing** — Optimizes batched routes to minimize distance, cost, and delivery windows

```
Order Created → Auto-Assigned → Route Optimized → Driver Notified → In Transit → Delivered
```

## Real-Time Fleet Tracking

Every driver running the SEFARI driver app reports their location in real time. From the console you can:

- View all active drivers on a live map
- See current order status per driver
- Send messages directly to drivers
- Monitor ETAs and flag delays

## Proof of Delivery

SEFARI captures a complete delivery record at every stop:

- **Digital signature** — Captured on the driver's mobile device
- **Photo confirmation** — Optional photo of the delivered item
- **Timestamp & geolocation** — Automatically recorded on delivery completion
- **Customer notification** — Automatic SMS/email sent to the recipient with confirmation

## Analytics & Reporting

Track performance across your entire operation:

| Metric | Description |
|--------|-------------|
| On-time rate | % of deliveries completed within the promised window |
| Route efficiency | Actual vs. optimal distance traveled |
| Driver performance | Orders per hour, completion rate, customer ratings |
| Cost per delivery | Fuel, labor, and operational cost per order |

<Callout>
All reports can be exported as CSV or accessed via the SEFARI API for integration with your BI tools.
</Callout>
